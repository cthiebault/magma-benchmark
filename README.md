magma-benchmark
===============

Benchmark for [Magma Datasources](https://github.com/obiba/magma)

Supports:

* Hibernate Datasource
    * MySQL
    * HsqlDB
    * MariaDB (TODO)
* JDBC Datasource
    * MySQL
    * HsqlDB
* MongoDB Datasource
* OrientDB Datasource (TODO)

To run `./gradlew run` and results in `results.csv`
